Follow-up Prompt: UI HUD Designer & PNG Skin Engine
Role: Senior UI/UX Engineer (PySide6 / Qt). Task: Implement a "HUD Skin" feature for the underwater media app that allows users to import a PNG graphic, adjust its transparency, and anchor dynamic telemetry data points on top of it.
Core Functional Requirements:
1. PNG Background (The "Skin"):
• Import: Allow the user to select a PNG file (e.g., a Shearwater or Garmin computer face).
• Opacity Control: A slider in the UI that dynamically updates the opacity (0.0 to 1.0) of the PNG QGraphicsPixmapItem.
• Scaling: The PNG must be resizable while maintaining aspect ratio, with its dimensions and position stored as percentages of the video frame.
2. Dynamic Data Anchoring (Text-on-PNG):
• Parent-Child Relationship: Logic to "attach" telemetry text items (Depth, Temp, etc.) to the PNG skin. If the user moves the PNG, the data points should move with it.
• Formatting: Ability to select font size, color, and weight for the text that sits on top of the PNG.
3. The JSON Layout Schema (Updated): The export logic must include a nested structure for the HUD:
JSON
{
  "hud_skin": {
    "path": "assets/shearwater_perdix.png",
    "opacity": 0.75,
    "scale": 0.25,
    "x_pct": 0.05,
    "y_pct": 0.80,
    "linked_elements": [
      {"field": "depth", "rel_x": 0.1, "rel_y": 0.2, "color": "#FFFFFF"},
      {"field": "temp", "rel_x": 0.1, "rel_y": 0.4, "color": "#00FF00"}
    ]
  }
}

4. FFmpeg "Filter Complex" Translation:
• The backend logic must take this JSON and generate a multi-layered FFmpeg command that can be used in the color.py
ColorCorrectionEngine
• Layer 1: Background Video.
• Layer 2: PNG Overlay with alpha/opacity applied via the format=yuva444p and lutrgb or colorchannelmixer filters.
• Layer 3+: drawtext filters with coordinates calculated by combining the HUD position and the element's relative position.
5. Data Sync:
• Update the text values in real-time based on the Pydantic Waypoint data corresponding to the current "Time Slider" position.
Output Requirements:
1. HUDManager Class: Implementation of the dragging and parenting logic in PySide6.
2. Opacity Slider Logic: A code snippet showing how to update the QGraphicsItem opacity in real-time.
3. FFmpeg Filter String Generator: A Python function that iterates through the linked_elements in the JSON and builds the final FFmpeg command for a multi-layered overlay.
